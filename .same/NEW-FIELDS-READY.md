## 🗄️ База данных

### SQL миграция

**Файл:** `.same/FINAL-MIGRATION.sql`

Этот файл содержит:
1. ✅ Статистику просмотров (views)
2. ✅ Новые поля автомобилей (transmission, engine, fuel_type, color)

**Выполните этот SQL в Supabase:**

```sql
-- Новые поля
ALTER TABLE cars ADD COLUMN IF NOT EXISTS transmission TEXT;
ALTER TABLE cars ADD COLUMN IF NOT EXISTS engine TEXT;
ALTER TABLE cars ADD COLUMN IF NOT EXISTS fuel_type TEXT;
ALTER TABLE cars ADD COLUMN IF NOT EXISTS color TEXT;

-- Индексы для фильтрации
CREATE INDEX IF NOT EXISTS idx_cars_transmission ON cars(transmission);
CREATE INDEX IF NOT EXISTS idx_cars_fuel_type ON cars(fuel_type);
CREATE INDEX IF NOT EXISTS idx_cars_color ON cars(color);
```

## 🧪 Как протестировать

### 1. Выполните SQL миграцию

Откройте: https://app.supabase.com/project/dybmmygpbyaawmiweidu/sql/new

Скопируйте весь SQL из `.same/FINAL-MIGRATION.sql` и выполните.

### 2. Добавьте тестовое объявление

1. Откройте админ-панель: http://localhost:3000/admin
2. Или откройте каталог и прокрутите вверх к форме
3. Заполните новые поля:
   - **Название:** `BMW X5 M50d 2020`
   - **Марка:** `BMW`
   - **Модель:** `X5`
   - **Год:** `2020`
   - **Коробка передач:** `Автомат`
   - **Двигатель:** `3.0 TDI 286 л.с.`
   - **Тип топлива:** `Дизель`
   - **Цвет:** `Черный металлик`
4. Добавьте фото
5. Нажмите "Добавить объявление"

### 3. Проверьте отображение

1. Откройте страницу детального просмотра
2. В секции "Характеристики" должны быть все новые поля
3. Проверьте, что данные отображаются корректно

### 4. Проверьте редактирование

1. Откройте админ-панель
2. Нажмите "Редактировать" на объявлении
3. Измените любое поле (например, цвет на "Белый")
4. Сохраните изменения
5. Проверьте, что изменения применились
