# 🎥 Как заменить видео на своё

## Где находится видео?
Откройте файл: `brabus-clone/src/components/Hero.tsx`

## Замените URL видео
Найдите строку 12-15:
```tsx
<source
  src="https://cdn.pixabay.com/video/2023/05/30/163856-831301062_large.mp4"
  type="video/mp4"
/>
```

## Варианты замены:

### 1. Использовать внешнюю ссылку
Замените URL на ссылку на ваше видео:
```tsx
<source
  src="https://ваша-ссылка-на-видео.mp4"
  type="video/mp4"
/>
```

### 2. Использовать локальное видео
1. Создайте папку `public/videos` в корне проекта
2. Поместите туда ваше видео (например, `hero-video.mp4`)
3. Замените src:
```tsx
<source
  src="/videos/hero-video.mp4"
  type="video/mp4"
/>
```

## Рекомендации для видео:
- **Формат**: MP4 (H.264)
- **Разрешение**: 1920x1080 или выше
- **Длительность**: 10-30 секунд (будет зацикливаться)
- **Размер файла**: До 10-15 МБ для быстрой загрузки
- **FPS**: 30
- **Соотношение сторон**: 16:9

## Видео будет:
✅ Автоматически воспроизводиться
✅ Зацикливаться
✅ Без звука (muted)
✅ На весь экран
✅ С затемнением для читаемости текста
