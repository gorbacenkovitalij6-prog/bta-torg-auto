## 🎨 Новые функции

### Множественная загрузка фото
- Форма поддерживает выбор нескольких файлов (`multiple` атрибут)
- Превью всех выбранных изображений в grid 2x4
- Возможность удалить превью перед отправкой (кнопка X)
- Все фото загружаются в Supabase Storage
- Массив URL сохраняется в поле `images` (JSONB)

### Галерея в карточках
- Стрелки для навигации по фото (появляются при hover)
- Счетчик фото (1/5)
- Плавные переходы между изображениями
- Zoom эффект при hover

### Фильтры и поиск
- **Поиск:** по названию, марке, модели, описанию (регистронезависимый)
- **Фильтры:**
  - Цена от/до
  - Год от/до
  - Марка (dropdown с уникальными значениями)
- Кнопка "Сбросить все"
- Счетчик найденных результатов
- Автоматический сброс пагинации при изменении фильтров

### Админ-панель
- **URL:** `/admin`
- **Аутентификация:** Через Supabase Auth (email + password)
- **Функции:**
  - Просмотр всех объявлений в таблице
  - Удаление объявлений (с подтверждением)
  - Выход из системы
- **Безопасность:**
  - Форма входа для неавторизованных
  - Проверка JWT токена через Supabase Auth
  - RLS политики на уровне БД

## 🔧 Технические заметки
- Проект работает на Next.js 15 с Turbopack
