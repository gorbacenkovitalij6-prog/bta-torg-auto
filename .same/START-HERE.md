## 📋 Что нужно сделать (всего 3 шага)

### Шаг 1: Настроить Supabase (5 минут)

**Откройте:** https://app.supabase.com/project/dybmmygpbyaawiweidu

**Выполните 3 действия:**

1. **Создать таблицу `cars`**
   - SQL Editor → New query
   - Скопировать весь SQL из `brabus-clone/.same/setup-database.sql`
   - Run (Ctrl+Enter)
   - ✅ Success. No rows returned

2. **Создать bucket `car-images`**
   - Storage → Create bucket
   - Name: `car-images`
   - Public bucket: ✅ ON
   - Create

3. **Создать политики Storage**
   - Storage → car-images → Policies
   - Если политик нет, выполните в SQL Editor:
   ```sql
   CREATE POLICY "Публичное чтение изображений"
     ON storage.objects FOR SELECT
     USING (bucket_id = 'car-images');

   CREATE POLICY "Разрешить загрузку изображений"
     ON storage.objects FOR INSERT
     WITH CHECK (bucket_id = 'car-images');
   ```

**Проверка:** В Table Editor должна быть таблица `cars`, в Storage должен быть bucket `car-images` (Public ON)

### Шаг 3: Протестировать загрузку (2 минуты)

1. **Откройте** http://localhost:3000/catalog

2. **Откройте консоль браузера** (F12)

3. **Заполните форму:**
   - **Название:** `BMW X5 M50d 2020`
   - **Марка:** `BMW`
   - **Модель:** `X5`
   - **Цена:** `4500000`
   - **Год:** `2020`

4. **Выберите 3 фото:**
   - Клик на поле "Фотографии"
   - Удерживайте Ctrl (Windows) или Cmd (Mac)
   - Выберите все 3 фото
   - Открыть

5. **Проверьте превью:**
   - Должны появиться 3 превью изображений
   - На каждом превью кнопка X при наведении

6. **Нажмите "ДОБАВИТЬ ОБЪЯВЛЕНИЕ"**

7. **Проверьте консоль браузера:**
   ```
   🚀🚀🚀 VERSION 215 - SERVER API UPLOAD 🚀🚀🚀
   📤 Uploading to server API
   ✅ Server returned: {url: '...'}
   ```

8. **Проверьте результат:**
   - Должно появиться: ✅ Объявление добавлено!
   - Карточка с автомобилем появляется ниже
   - Наведите на карточку → появятся стрелки для навигации по фото
   - Счетчик: "1 / 3"

## 🐛 Если что-то не работает

### "Ошибка загрузки"

1. **Проверьте консоль браузера (F12)** → скопируйте текст ошибки
2. **Проверьте терминал dev сервера** → ищите ERROR
3. **Возможные причины:**
   - `Bucket not found` → создайте bucket `car-images`
   - `Policy violation` → создайте политики Storage
   - `Supabase not configured` → проверьте `.env`

### "Изображения не отображаются"

1. Проверьте что bucket `car-images` публичный (Public ON)
2. Перезапустите dev сервер (Ctrl+C → `bun run dev`)
3. Проверьте `next.config.js` → должен быть `dybmmygpbyaawiweidu.supabase.co`
