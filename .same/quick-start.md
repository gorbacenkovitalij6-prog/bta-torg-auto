# 🚀 БТА Торг - Быстрый старт

## Шаг 1: Настройка Supabase (5 минут)

### 1.1 Создайте таблицу `cars`

1. Откройте Supabase Dashboard: https://app.supabase.com/project/dybmmygpbyaawiweidu
2. Перейдите в **SQL Editor**
3. Скопируйте весь SQL из файла `.same/setup-database.sql`
4. Вставьте в редактор и нажмите **RUN**
5. Должно появиться сообщение: `Success. No rows returned`

### 1.2 Создайте Storage bucket

1. Перейдите в **Storage** → **Create bucket**
2. Введите имя: `car-images`
3. **Public bucket:** ✅ Включить
4. Нажмите **Create bucket**

### 1.3 Настройте Storage политики

Перейдите в **Storage** → `car-images` → **Policies** и создайте:

**Политика 1: Публичное чтение**
```sql
CREATE POLICY "Публичное чтение изображений"
  ON storage.objects FOR SELECT
  USING (bucket_id = 'car-images');
```

**Политика 2: Разрешить загрузку**
```sql
CREATE POLICY "Разрешить загрузку изображений"
  ON storage.objects FOR INSERT
  WITH CHECK (bucket_id = 'car-images');
```

### 1.4 Создайте администратора

1. Перейдите в **Authentication** → **Users**
2. Нажмите **Add user** → **Create new user**
3. Заполните:
   - **Email:** ваш email (например, `admin@btatorg.ru`)
   - **Password:** придумайте надежный пароль
   - **Auto Confirm User:** ✅ Включить
4. Нажмите **Create user**

✅ **Supabase настроен!**

## Шаг 2: Проверка работы (2 минуты)

### 2.1 Откройте каталог

1. Перейдите на http://localhost:3000/catalog
2. Вы должны увидеть форму добавления объявления

### 2.2 Добавьте тестовое объявление

1. Заполните форму:
   - **Название:** `BMW X5 2020`
   - **Марка:** `BMW`
   - **Модель:** `X5`
   - **Цена:** `4500000`
   - **Год:** `2020`
   - **Пробег:** `45000`
   - **Описание:** `Отличное состояние`

2. Выберите 2-3 фото (Ctrl+Click или Cmd+Click)
   - Можете скачать тестовые фото с Unsplash
   - Или использовать любые фото автомобилей с компьютера

3. Нажмите **Добавить объявление**

4. Проверьте:
   - Должно появиться сообщение "Объявление добавлено!"
   - Карточка автомобиля появится ниже
   - При наведении на карточку появятся стрелки для навигации по фото

### 2.3 Проверьте фильтры

1. Добавьте еще 1-2 объявления с разными характеристиками
2. Попробуйте:
   - Поиск по названию (введите "BMW" в строку поиска)
   - Фильтр по марке (выберите марку из выпадающего списка)
   - Фильтр по цене (введите "от 2000000" и "до 5000000")
   - Нажмите "Сбросить все"

### 2.4 Проверьте админ-панель

1. Откройте http://localhost:3000/admin
2. Войдите с email и паролем администратора
3. Вы должны увидеть таблицу с объявлениями
4. Попробуйте удалить объявление (нажмите кнопку "Удалить" → подтвердите)
5. Выйдите из системы (кнопка "Выйти" вверху справа)

✅ **Всё работает!**

## Шаг 3: Если что-то не работает

### Проблема: "Ошибка загрузки"

**Решение:**
1. Откройте консоль браузера (F12)
2. Проверьте логи в консоли:
   ```
   🚀🚀🚀 VERSION 213 - SERVER API UPLOAD 🚀🚀🚀
   ```
3. Если нет логов → проверьте `.env` файл (должен содержать `SUPABASE_SERVICE_ROLE_KEY`)
4. Если ошибка "Bucket not found" → проверьте, что бакет `car-images` создан

**Проверка в терминале:**
В терминале где запущен `bun run dev` должны быть логи:
```
=== UPLOAD API CALLED ===
ENV: { url: '...', hasServiceKey: true, adminExists: true }
Uploading to Supabase: ...
Upload success! Path: ...
```

### Проблема: "Не могу войти в админ-панель"

**Решение:**
1. Проверьте, что пользователь создан в Supabase Authentication
2. Проверьте, что галочка "Auto Confirm User" была включена
3. Попробуйте сбросить пароль в Supabase Dashboard
4. Проверьте консоль браузера на ошибки (F12)

### Проблема: "Фильтры не работают"

**Решение:**
1. Убедитесь, что добавлено минимум 2 объявления
2. Проверьте, что заполнены поля Brand, Year, Price
3. Проверьте консоль браузера на ошибки JavaScript

### Проблема: "Изображения не отображаются"

**Решение:**
1. Проверьте `next.config.js` → должен содержать `dybmmygpbyaawiweidu.supabase.co` в remotePatterns
2. Перезапустите dev server (Ctrl+C → `bun run dev`)
3. Проверьте, что bucket `car-images` настроен как публичный
